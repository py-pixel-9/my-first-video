====================================================================
🔧 컴파일 방법:
====================================================================

1. MT4 열기

2. MetaEditor 열기 (F4 또는 Tools → MetaQuotes Language Editor)

3. File → Open → SMC_Professional_Signal.mq4 선택

4. Compile 버튼 클릭 (F7 또는 Compile 아이콘)

5. 하단에 "0 error(s), 0 warning(s)" 확인

6. MT4/MQL4/Indicators/ 폴더에 SMC_Professional_Signal.ex4 생성됨

====================================================================
📊 차트에 적용:
====================================================================

1. MT4 Navigator 열기 (Ctrl + N)

2. Indicators → Custom → SMC_Professional_Signal 찾기

3. 차트에 드래그 앤 드롭

4. 설정 창에서 파라미터 확인:
   - Time-Horizon: 35 (기본값)
   - Minimum Price Range: 30.0 pips
   - Max Swings: 5개
   - Breakout Distance: 30.0 pips

5. OK 클릭

====================================================================
🎯 옵션 B 주요 특징:
====================================================================

✅ 최소 가격 범위 필터 (30 pips)
   → 너무 작은 스윙은 무시

✅ 최근 5개 스윙만 추적
   → 의미 있는 구조만 감지

✅ 돌파 거리 필터 (30 pips)
   → 충분한 돌파만 인정

✅ 추세 필터 (Higher High/Low 패턴)
   → BOS와 CHoCH 정확히 구분

✅ 시각적 개선
   - 굵은 선 (너비 2)
   - 큰 텍스트 (Arial Black)
   - 명확한 색상 구분

4. Start 클릭

5. 차트 보기 → 인디케이터 확인

→ 현재 추세: 상승

❓ 문제 해결:
====================================================================

Q: 신호가 하나도 안 나와요
A: Time-Horizon을 낮추세요 (35 → 30 → 25)

Q: 신호가 여전히 너무 많아요
A: Breakout Distance를 높이세요 (30 → 40 → 50)

Q: 컴파일 오류가 나요
A: MT4 버전 확인 (Build 1260 이상 필요)

Q: 백테스트에서 안 보여요
A: Visual Mode를 켜고 과거 캔들에서 확인
