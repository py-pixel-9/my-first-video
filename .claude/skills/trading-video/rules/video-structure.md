# Video Structure Template

## 트레이딩 영상 기본 구조

### 1. 훅 (0-15초)
- 성과 수치 또는 충격적 결과로 시작
- "이 EA가 $10,000을 $21,000으로 만들었습니다"
- Remotion: 숫자 카운트업 애니메이션 + 수익률 그래프

### 2. 문제 제기 (15-45초)
- 시청자의 공감을 이끌어내는 문제점
- "수동 매매로 지치셨나요?"
- Remotion: 텍스트 애니메이션 + 아이콘

### 3. 전략 설명 (45초-3분)
- 핵심 전략 단계별 설명
- 차트 위에 진입/청산 포인트 표시
- Remotion: 캔들차트 애니메이션 + 화살표 + 레벨 표시

### 4. 백테스트 결과 (3-4분)
- 실제 백테스트 성과 데이터
- 승률, 수익률, 최대 낙폭
- Remotion: 바 차트/파이 차트 애니메이션

### 5. 설정 가이드 (4-5분)
- MT4 설치 및 EA 적용 방법
- 파라미터 설명
- Remotion: 화면 녹화 + 오버레이 텍스트

### 6. CTA (5-5:30분)
- 구독/좋아요 유도
- 다음 영상 예고
- Remotion: 구독 버튼 애니메이션

## 대본 작성 규칙
- 문장은 짧고 명확하게
- 전문용어는 반드시 설명 포함
- 한국어 기준, 1분당 약 150-180자
- 자막은 2줄 이내, 한 줄 15자 이내
