# Trading Terms Reference

## 주요 트레이딩 용어 (한국어 ↔ 영어)

### 주문 유형
| 한국어 | 영어 | 설명 |
|--------|------|------|
| 시장가 주문 | Market Order | 현재 가격에 즉시 체결 |
| 지정가 주문 | Limit Order | 지정 가격에 도달 시 체결 |
| 바이스탑 | Buy Stop | 현재가 위에 매수 대기 |
| 셀스탑 | Sell Stop | 현재가 아래에 매도 대기 |
| 손절 | Stop Loss (SL) | 손실 제한 주문 |
| 익절 | Take Profit (TP) | 이익 실현 주문 |
| 트레일링 스탑 | Trailing Stop | 가격 추종 손절 |

### 전략 용어
| 한국어 | 영어 | 설명 |
|--------|------|------|
| 돌파 | Breakout | 지지/저항 돌파 |
| 되돌림 | Pullback/Retracement | 추세 중 반대 방향 움직임 |
| 지지선 | Support | 가격 하락 방어 레벨 |
| 저항선 | Resistance | 가격 상승 방어 레벨 |
| 추세 | Trend | 가격 방향성 |
| 횡보 | Range/Sideways | 방향 없는 움직임 |
| 변동성 | Volatility | 가격 변동 크기 |

### EA/알고리즘 용어
| 한국어 | 영어 | 설명 |
|--------|------|------|
| 전문가 자문 | Expert Advisor (EA) | MT4 자동매매 프로그램 |
| 백테스트 | Backtest | 과거 데이터 검증 |
| 포워드테스트 | Forward Test | 실시간 데모 검증 |
| 최적화 | Optimization | 파라미터 최적값 탐색 |
| 룩백 기간 | Lookback Period | 계산에 사용할 과거 봉 수 |
| 매직넘버 | Magic Number | EA 주문 식별 번호 |
| 슬리피지 | Slippage | 주문가와 체결가 차이 |
| 스프레드 | Spread | 매수/매도 호가 차이 |

### SMC (Smart Money Concepts)
| 한국어 | 영어 | 설명 |
|--------|------|------|
| 스윙 하이 | Swing High | 좌우 봉보다 높은 고점 |
| 스윙 로우 | Swing Low | 좌우 봉보다 낮은 저점 |
| 유동성 | Liquidity | 주문이 집중된 가격대 |
| 주문블록 | Order Block | 기관 매집/분배 영역 |
| 공정가치갭 | Fair Value Gap (FVG) | 빠른 움직임으로 생긴 가격 갭 |

### 리스크 관리
| 한국어 | 영어 | 설명 |
|--------|------|------|
| 리스크 비율 | Risk Percentage | 계좌 대비 위험 비율 |
| 손익비 | Risk/Reward Ratio | 손절:익절 비율 |
| 최대 낙폭 | Max Drawdown | 최대 자금 감소폭 |
| 포지션 사이징 | Position Sizing | 진입 수량 결정 |
