# Trading Video Content Creation Skill

트레이딩 유튜브 영상 제작을 위한 종합 스킬

## 워크플로우

1. **리서치** → 시장 분석, 전략 조사
2. **기획/대본** → 영상 구조, 스크립트 작성
3. **Remotion 영상** → 차트 애니메이션, 자막, 설명 영상 자동 생성
4. **편집** → 유현호님 수동 편집

## 규칙 파일

Read individual rule files for detailed explanations:

- `rules/chart-animations.md` - 캔들차트, 라인차트 애니메이션
- `rules/trading-terms.md` - 트레이딩 용어 한영 매핑
- `rules/video-structure.md` - 영상 구조 및 대본 템플릿
- `rules/ea-explanation.md` - EA/알고리즘 트레이딩 설명 가이드
